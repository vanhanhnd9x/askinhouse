== Description ==

WP-Vietnamese-URL là plugin giúp chuyển đổi các chuỗi tiếng Việt có dấu trong tiêu đề, category và tag thành chuỗi không dấu một cách chuẩn xác nhất.

* Nếu bạn thấy plugin này có ích, hãy donate tác giả một ly cà phê nhé ?

== Installation ==

Rất đơn giản, download trực tiếp từ WordPress, kích hoạt và sử dụng
